## Avaliação de Lógica de Programação 1° ano

**1. De acordo com os conceitos de Lógica de Programação, o que é um algoritmo?**

a) Um componente físico (hardware) responsável pelo processamento do computador.

b) Uma linguagem de programação específica usada para criar fluxogramas.
c) Uma sequência de passos lógicos para resolver um problema.
d) Uma variável utilizada para armazenar números inteiros.

**2. Na representação visual de algoritmos utilizando fluxogramas, qual forma geométrica é usada para indicar um "Desvio condicional" (decisão)?**
a) Retângulo
b) Losango
c) Círculo (Terminador)
d) Paralelogramo

**3. Em muitas linguagens de programação e em pseudocódigo, qual operador relacional é utilizado para verificar se um valor é "Diferente" de outro?**
a) ==
b) <=
c) !=
d) >=

**4. Segundo a tabela verdade do operador lógico "E" (AND), o resultado final só será VERDADEIRO em qual das condições a seguir?**
a) Quando pelo menos uma das proposições for verdadeira.
b) Quando a primeira proposição for verdadeira, independentemente da segunda.
c) Quando ambas as proposições (p e q) forem falsas.
d) Quando ambas as proposições (p e q) forem verdadeiras.

**5. Segundo a tabela verdade do operador lógico "OU" (OR), em qual cenário o resultado da expressão lógica será FALSO?**
a) Quando ambas as proposições (p e q) forem falsas.
b) Quando uma proposição for verdadeira e a outra falsa.
c) Quando ambas as proposições (p e q) forem verdadeiras.
d) Sempre que a segunda proposição for falsa.

**6. Ao criar uma variável para armazenar o valor exato "-45.999", qual Tipo Primitivo deve ser escolhido?**
a) Inteiro
b) Lógico
c) Caractere
d) Real

**7. Seguindo as regras de nomenclatura de variáveis (Guia de Identificadores), qual das opções abaixo apresenta um nome VÁLIDO para uma variável?**
a) 1NOTA
b) NOME CLIENTE
c) PRECO_TOTAL
d) TAXA%

**8. Qual é a principal diferença entre o uso de uma "Variável" e de uma "Constante" na lógica de programação?**
a) Variáveis podem mudar de valor durante a execução do programa, enquanto o valor da constante não muda nunca.
b) Constantes são numéricas e variáveis só armazenam texto.
c) Variáveis só existem em pseudocódigo, constantes apenas em fluxogramas.
d) Constantes recebem nomes aleatórios, variáveis exigem palavras reservadas.

**9. No fluxograma, o símbolo de Atribuição (um retângulo) é utilizado especificamente para:**
a) Receber dados através do teclado.
b) Exibir dados na tela do computador.
c) Indicar cálculos e atribuição de valores a variáveis.
d) Indicar o início do algoritmo.

a) O valor deve ser escrito em letras maiúsculas obrigatoriamente.
b) O valor deve estar contido entre aspas (duplas ou simples).
c) O valor deve terminar sempre com um ponto e vírgula dentro do texto.
d) O texto só pode conter letras do alfabeto, excluindo números ou espaços.
